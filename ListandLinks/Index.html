Step by Step instructions on how to create a new webpage on the production server

Step 1: Open Terminal which is an app in the bottom left corner of the screen with an icon that looks something like this: [>_ ]

Step 2: While in the Terminal, create a html file and a css file: index.html, main.css. to create this, type in gedit (index.html and/or main.css).

Step 3: After creating those two files, type in at the top , then put an opening an closing tab, (opening) (closing).

Step 4: Inside the html tags, add a head and a body tag which is similar to the html tag, except you replace html with head and body. make sure to that the tags are tabbed once to the right from the previous line.

Step 5: After adding the head and the body, you can create more tags within them such as title, which can be placed within the tag head. P for paragraph and h1 fro heading 1 cn be placed within the body tag which will then be shon on the website from the development server. If you want to view your website, you must start up the HTTP server by typing in python -m SimpleHTTPServer into the terminal, once it gets started, it will say Serving HTTP on (server). to view the website onlne, you need to type in localhost:8000 into the search bar. Once opened, the website should look blank because you haven't added anything to it yet.
